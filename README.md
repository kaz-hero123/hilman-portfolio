# Hilman Nidal Hamzi — Personal Portfolio 🚀

> A brutalist, engineering-first personal portfolio built to showcase backend projects, project management skills, and professional experience. Designed with a focus on sharp typography, premium micro-interactions, and high performance.

## ⚡ Key Features

- **Brutalist Aesthetic**: Clean, monochrome design with sharp edges, prioritizing content and readability over unnecessary fluff.
- **Buttery Smooth Scrolling**: Powered by [Lenis](https://github.com/studio-freight/lenis) for a native, momentum-based scrolling experience.
- **Premium Micro-Interactions**:
  - **Magnetic Buttons**: CTAs that subtly pull towards the cursor on hover.
  - **3D Tilt Cards**: Project cards that react to mouse movement with 3D parallax depth.
  - **Dynamic Timeline**: A scroll-linked progress line that lights up as you read the history section.
  - **Blur Reveals & Scramble Text**: Cinematic text decryption and focal reveals using [Framer Motion](https://www.framer.com/motion/).
  - **Infinite Marquee**: Auto-scrolling, seamlessly looping logo cloud for the technology stack.
- **Cinematic Noise**: A subtle SVG film grain overlay that gives the site an organic, tactile feel.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router, React)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scroll**: Lenis
- **Icons**: [Lucide React](https://lucide.dev/) & [Devicon](https://devicon.dev/)
- **Typography**: Google Fonts (Playfair Display, Pacifico, Inter, JetBrains Mono)

## 💻 Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/kaz-hero123/hilman-portfolio.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the result.
